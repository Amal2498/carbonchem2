Химические свойства алкенов

1️Гидрогалогенирование
CH₂=CH₂ + HCl → СH₃-CH₂-Cl

2 Гидратация
Условие: H₂SO₄/H₃PO₄, t°C
CH₂=CH₂ + H₂O → СH₃-CH₂-OH ( t°C, H⁺)

🥕 Правило Марковникова:
водород присоединяется к наиболее гидрированному атому углерода (даём богатым)
CH₂=CH-СH₃ + HCl → СH₃-CH(Сl)-СH₃

❌🥕 Правило Марковникова НЕ выполняется, если:
1. Реакция протекает в присутствии перекиси (Н₂О₂):
CH₂=CH-СH₃ + HCl → СH₂(Cl)-CH₂-СH₃ (Н₂О₂)

2. Наличие электроноакцепторных групп (-СF₃, -COOH)
CH₂=CH-COOH + HCl → Cl-CH₂-CH₂-COOH

3️ Галогенирование
CH₂=CH₂ + Cl₂ → Cl-СH₂-CH₂-Cl
Алкены реагируют с раствором хлора и брома в воде — хлорной и бромной водой соответственно.
В ходе реакции происходит обесцвечивание бромной воды!

4️ Гидрирование
Условие: t°C, Ni
CH₂=CH₂ + H₂ → СH₃-CH₃ ( t°C, Ni)

5️ Полимеризация
Условие: t°C, кат.
nCH₂=CH₂ → (-CH₂-CH₂-)n (t°C, кат.)
nCH₂=CH-СH₃ → (-CH₂-CH(СH₃)-)n (t°C, кат.)

6️ Каталитическое окисление
Условие: t°C, PdCl₂
2CH₂=CH₂ + O₂ → 2СH₃-CHO
2CH₂=CH-СH₃ + O₂ → 2CH₃-C(О)-СH₃

7️ Мягкое окисление водным раствором KMnO₄ (реакция Вагнера)
Условие: 0-20°C, KMnO₄ (Н₂О)
3CH₂=CH₂ + 2KMnO₄ + 4Н₂О → 3HO-СH₂-CH₂-OH + 2MnO₂ + 2KOH (0-20°C)
Происходит разрыв пи-связи, к каждому из двух углеродов присоединяется -OH группа. Образуются двухатомные спирты (гликоли).

8️ Жёсткое окисление разберём в следующем шаге

9️ Галогенирование (реакция замещения)
Условие: t°C
CH₂=CH-СH₃ + Сl₂ → CH₂=CH-СH₂-Cl + HCl (t°C)
Двойная связь остаётся нетронутой, происходит замещение в α-положение

1️0 Горение (полное окисление)
С₂H₄ + 3O₂ → 2CO₂ + 2H₂O (t°C)
